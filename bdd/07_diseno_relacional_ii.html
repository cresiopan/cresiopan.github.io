Algoritmo de cubrimiento minimal Cobertura y equivalencia Definiciones decimos que el conjunto F cubre a G cuando toda dependencia funcional X → Y ∈ G puede ser inferida a partir de F . Es decir: ∀(X → Y ) ∈ G : F |= X → Y . Dos conjuntos de dependencias funcionales F y G son equivalentes cuando cada uno de ellos es cubierto por el otro. En coinciden: F

= G

. Lo simbolizaremos F ≡ G. Ejercicio F 1 = { A → B, B → C, C → A} y F 2 = { A → C, C → B, B → A} son equivalentes.
